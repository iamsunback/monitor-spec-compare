# 📖 คู่มือการใช้งานระบบเปรียบเทียบสเปคสินค้า (Monitor Spec Comparison)

## 🎯 วัตถุประสงค์ของระบบ

ระบบถูกออกแบบมาเพื่ออำนวยความสะดวกในการตรวจสอบและเปรียบเทียบข้อมูลจำเพาะ (Specification) ของสินค้า โดยเน้นการใช้งานผ่าน **"สมาร์ทโฟนแนวตั้ง"** เป็นหลัก เพื่อให้พนักงานหน้าร้าน พนักงานคลังสินค้า หรือแม้แต่ลูกค้า สามารถค้นหา สแกนบาร์โค้ด และเปรียบเทียบสเปคสินค้ารุ่นต่างๆ ได้อย่างรวดเร็ว แม่นยำ และใช้งานได้ทันทีโดยไม่ต้องติดตั้งแอปพลิเคชันหรือทำการ Log in

---

## ✨ ความสามารถหลักของโปรแกรม (Key Features)

**1. ระบบค้นหาอัจฉริยะ (Smart Search)**

* สามารถค้นหาสินค้าได้จากการพิมพ์ข้อความเพียงบางส่วน
* รองรับการค้นหาจากหลากหลายข้อมูล ได้แก่ รหัสสินค้า (CODE), บาร์โค้ด (Barcode), หมายเลขชิ้นส่วน (PartNo) หรือ รายละเอียดสินค้า (Description)

**2. ระบบสแกนผ่านกล้องสมาร์ทโฟน (Camera Scanner)**

* **สแกน Barcode ข้างกล่อง:** สามารถใช้กล้องมือถือสแกนบาร์โค้ดที่ตัวสินค้าเพื่อเรียกดูสเปคได้ทันที
* **สแกน QR Code (Direct Link):** รองรับการสแกน QR Code ที่ถูกสร้างจากระบบ เพื่อดึงรหัสสินค้ามาประมวลผล
* **ระบบ Popup ยืนยันความถูกต้อง:** เมื่อสแกนพบสินค้า ระบบจะแสดงหน้าต่าง Popup สรุปข้อมูลเบื้องต้น (CODE, Barcode, Description) ให้ผู้ใช้ตรวจสอบก่อนกดปุ่ม "Add to Compare" เพื่อป้องกันความผิดพลาด

**3. ตารางเปรียบเทียบข้อมูล (Comparison Table)**

* รองรับการเปรียบเทียบสเปคสินค้าพร้อมกันได้สูงสุด **4 รุ่น**
* **Responsive Design:** ออกแบบมาเพื่อมือถือโดยเฉพาะ โดยตารางจะทำการ "ตรึง (Sticky)" คอลัมน์หัวข้อสเปคไว้ด้านซ้ายเสมอ ทำให้ผู้ใช้สามารถปัดหน้าจอเลื่อนดูข้อมูลเปรียบเทียบซ้าย-ขวาได้โดยไม่งง
* สามารถกดปุ่มกากบาท (X) เพื่อลบหรือเปลี่ยนรุ่นสินค้าที่ต้องการเปรียบเทียบได้ตลอดเวลา

**4. การปรับแต่งหัวข้อการแสดงผล (Customizable Specs)**

* ผู้ใช้สามารถเลือกซ่อน (Hide) หรือแสดง (Show) หัวข้อสเปคต่างๆ ในตารางได้ตามต้องการ เพื่อลดความซับซ้อนของข้อมูลและเน้นดูเฉพาะจุดที่สนใจ
* *หมายเหตุ: หัวข้อสำคัญได้แก่ CODE, Description และ PartNo จะถูกบังคับแสดงเสมอเพื่อไม่ให้สับสน*

**5. ระบบสร้างและดาวน์โหลด QR Code ประจำรุ่น (Direct Link QR Generator)**

* เมื่อผู้ใช้เลือกดูสินค้าเพียง 1 รุ่น ระบบจะสร้าง QR Code สำหรับสินค้ารุ่นนั้นขึ้นมาให้อัตโนมัติ
* มีปุ่ม **"Save as PNG"** เพื่อดาวน์โหลดรูป QR Code ไปใช้งานต่อได้ทันที (ระบบจะตั้งชื่อไฟล์เป็นรหัส CODE ของสินค้าให้โดยอัตโนมัติ)
* นำ QR Code นี้ไปพิมพ์แปะไว้ที่ป้ายราคาหรือป้ายหน้าชั้นวางสินค้า เมื่อมีคนนำมือถือมาสแกน ระบบจะพาเข้าเว็บและแสดงข้อมูลของสินค้ารุ่นนั้นเพียงรุ่นเดียวทันที

**6. ฐานข้อมูลออนไลน์แบบ Real-time (Google Sheets Integration)**

* ระบบเชื่อมต่อดึงข้อมูลโดยตรงจาก Google Sheets
* ผู้ดูแลระบบสามารถเพิ่มสินค้าใหม่ แก้ไขสเปค หรือลบสินค้าใน Google Sheets ได้ตามปกติ ข้อมูลบนหน้าเว็บไซต์จะอัปเดตตามโดยอัตโนมัติ (เมื่อกดรีเฟรชหน้าจอ) โดยไม่ต้องแก้โค้ดโปรแกรมใดๆ

## 📱 ข้อแนะนำเพิ่มเติมสำหรับการใช้งาน

* **การขอสิทธิ์เข้าถึงกล้อง:** ในการใช้งานปุ่ม "📷 สแกน" ครั้งแรกบนสมาร์ทโฟน เบราว์เซอร์จะเด้งข้อความขออนุญาตเข้าถึงกล้องถ่ายรูป ผู้ใช้จำเป็นต้องกด **"อนุญาต" (Allow)** จึงจะสามารถใช้งานระบบสแกนได้
* **การสแกนในที่มืด:** หากสแกนบาร์โค้ดแล้วระบบไม่ตอบสนอง ให้ตรวจสอบแสงสว่างบริเวณที่สแกน หรือปรับระยะห่างของกล้องกับบาร์โค้ดให้พอดี
